# Проект: WayHome (Построение маршрутов между городами с использованием Yandex API)

**WayHome** — это консольное приложение на C++ для поиска маршрутов между городами с использованием API [Яндекс Расписания](https://yandex.ru/dev/rasp/). Программа выводит расписание поездки и полностью кэширует данные для минимизации запросов к API.

---

## Что реализовано в проекте

### Работа с библиотеками
*   **HTTP-запросы** через библиотеку **cpr** (C++ Requests)
*   **Парсинг JSON** с помощью **nlohmann/json**

### Кэширование
*   **CacheCities** — список всех городов обновляется раз в месяц
*   **CacheTrips** — расписания маршрутов кэшируются на сутки
*   Автоматическое ограничение размера кэша (максимум 20 файлов с поездками)
*   Удаление старых кэшей, по аналогичному lru-cache способу

### Интерактивный интерфейс
*   Интерфейс, понятный любому пользователю
*   Вывод списка похожих названий городов (в случае, если пользователь ошибся с названием)
*   Понятный вывод - таймлайн с информацией о длительности поездок, местах пересадок
*   Красивое оформление вывода: вся информация на расском, используются цветной текст
*   Возможность изменить дату и количество пересадок в случае, когда не найдено подходящих маршрутов
*   Работа с несколькими запросами за один запуск программы
*   Гарантия, что программа не упадет при неправильных запросах пользователя или внешних факторах

---

## Примеры использования

### Запуск с указанием даты
```bash
./WayHome --date=2026-03-28
```

### Интерактивный режим
```bash
./WayHome
```
**Интерфейс:**

```bash
Город отправления: Новороссийск

Город прибытия: ВлАдИвОсТоК

Дата поездки (например 28.03.2026, сегодня, завтра): завтра 

Максимальное количество пересадок:
  1. Только прямые
  2. Не более 1 пересадки
  3. Не более 2 пересадок
  4. Без ограничений
Ваш выбор (1-4): 4

Обновляем список возможных маршрутов...

Найдено маршрутов: 10

Сколько показать? (1-10): 3
```
